# Exiban

Library for manipulating and validating IBAN account numbers.

## Installation

The package can be installed as:

  1. Add exiban to your list of dependencies in `mix.exs`:

        def deps do
          [{:exiban, "~> 0.0.1"}]
        end


  2. Run `mix deps.get` in your console to fetch from Hex
